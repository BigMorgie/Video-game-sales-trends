# Historical Video Game Trends


### This project was made to see what sort of trends have been happening in video game sales since they started releasing them in the 1980's

#### The dataset I used was from Kaggle and can be located with this provided link 
https://www.kaggle.com/datasets/volodymyrpivoshenko/video-game-sales-dataset 

### My objectives for this project was to clean and wrange the data ayalyze it for any significant trends.

#### Instructions on how to run the project files.
1. Clone repository.
2. Create a virtual environment
3. Install the required Python packages:  
   pip install -r requirements.txt
